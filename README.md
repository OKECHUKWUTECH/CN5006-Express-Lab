# CN5006 Week 4 Lab – Node.js Express REST API

**Module:** CN5006 – Web and Mobile Development  
**Week:** 4 

---

## **Overview**
This project demonstrates how to create a basic **REST API** using **Node.js** and **Express**. The lab focused on:

- Setting up an Express server
- Creating GET and POST routes
- Handling dynamic route parameters
- Reading and querying a JSON file
- Serving HTML forms and handling form submissions
